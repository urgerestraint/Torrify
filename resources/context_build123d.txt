## Common Patterns

### Semantic Selection (CRITICAL - Never use index-based selection)
part.faces().sort_by(Axis.Z).last  # Top face
part.edges().filter_by(Axis.Z)     # Vertical edges
part.faces().filter_by(GeomType.PLANE)  # Flat faces
part.edges().group_by(Axis.Z)[0]   # Bottom edge group

### Builder Pattern
with BuildPart() as part:
    with BuildSketch(Plane.XY):
        Rectangle(width, height, align=(Align.CENTER, Align.CENTER))
    extrude(amount=depth)
result = part.part

### Alignment
# Always use align parameter instead of manual centering math
Rectangle(w, h, align=(Align.CENTER, Align.CENTER))
Box(w, h, d, align=(Align.CENTER, Align.CENTER, Align.MIN))
