# Parameter Slider Tab (PRO)

The Parameter Slider tab is now available as a **PRO-only** feature.

It provides a code-free workflow for users who prefer visual controls over editing CAD code directly.

## What It Does

- Adds a `Parameters` tab next to the `Code` tab in the editor area.
- Lets users adjust editable parameters using sliders and input controls.
- Updates the model configuration and preview while they iterate.
- Keeps advanced users free to switch back to `Code` at any time.

## Why It Matters

This feature is designed for makers and designers with an aversion to code who still want fast iteration and parametric control.

## Availability

- `Parameters` is a **PRO feature**.
- Users without PRO can see the tab for discoverability, but access is locked.
- Activate PRO in `Settings > AI Configuration` using Gateway + license key.

## Best Results

Generate or structure models with a clear `CONFIGURATION (Editable Parameters)` section so parameter controls are easy to expose and adjust.
