# Torrify Documentation

Torrify is an AI-assisted desktop IDE for parametric CAD using OpenSCAD and build123d.

## Start Here

If you are new to Torrify, use this order:

1. [Install Torrify](./getting-started/installation.md)
2. [Run your first project](./getting-started/QUICKSTART.md)
